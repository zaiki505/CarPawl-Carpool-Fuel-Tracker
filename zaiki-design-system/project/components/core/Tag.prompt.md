Tiny uppercase category pill used on project cards; each category has its own hue.

```jsx
<Tag category="web">Web Development</Tag>
<Tag category="system">System Design</Tag>
<Tag category="personal">Personal Project</Tag>
```

Categories: `web` (blue), `system` (purple), `creative` (orange), `personal` (green). 0.6rem bold uppercase, 999px radius.
